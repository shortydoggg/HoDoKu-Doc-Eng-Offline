User manual

HoDoKu was designed to be as intuitive as possible. As with any program intuition alone would probably make you miss a few of the advanced features. This manual should help you with that. The manual is broken up in several pages. Every page contains a table of contents at the start of the page for easier navigation. If you find errors/omissions or simply have additional questions please use the support forum.

Happy solving!
